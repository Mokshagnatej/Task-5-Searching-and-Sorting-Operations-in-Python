5.2 Library Book Sorting and Search

Scenario: A librarian wants to sort book titles alphabetically to make searching easier.

Algorithm:

1. Store list of book titles.

2. Sort the list alphabetically.

3. Input book name to search.

4. Use binary search:

   • Check middle element.

   • If equal, found; else go left/right half.

6. Print result.

output:
sorted book list: ["Python Basics", "Data Structures", "Algorithm", "C Programming", "Database System", "Operating Systems", "Artificial Intelligence"]
