5.3 Online Shopping Price Filter

Scenario: An e-commerce platform has prices of products stored in a list. The company wants to display products sorted in ascending order of price so users can easily compare.

Algorithm:

1. Store product prices in a list.

2. Sort the list in ascending order.

3. Input price to search.

4. Use binary search for price. If found, confirm; else not found.

output:
Sorted prices: [1000, 1200, 1500, 2500]
Enter price to search: 1500
Price 1500 found at position 3
